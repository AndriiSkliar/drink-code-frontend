## Getting Started

To get started with AutoPtoCat locally, follow these steps:

### Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/AndriiSkliar/autoprocat
   ```

2. **Navigate to the project directory:**

   ```bash
   cd autoprocat
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```
